Bem-vindo ao GiltHub

Minha pequena versão do GitHub

" A palavra alemã gilt é a 3ª pessoa do singular do verbo gelten, que significa vale, é válido, aplica-se ou passa por. Ela indica validade (de documentos, regras) e também é usada para falar de reputação."

Use a barra de pesquisa para buscar um usuário do GitHub e ver a sua reputação por lá.
